# HW1

## Run the project

If you only want to run the project, you can follow the steps below.

### 1. Install bootstrap

```bash
npm i bootstrap@5.3.2
```

### 2. Install dependencies

```bash
cd backend
yarn
```

### 3. Run the server

```bash
yarn start
```

### 4. Open the frontend

Open `frontend/index.html` by clicking it in your file explorer.
